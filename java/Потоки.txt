Для реализации расширяем класс Thread или реализуем итерфейс Runnable
Методы Thread
	getName - получить имя потока
	getPriority - получить приоритет потока
	isAlive - выполняется поток или нет
	join - ожидать завершения потока
	run - входная точка потока
	sleep- приостановить выполение потока
	start - запустить метод Run() потока

static Thread currentThread()
	возвращает ссылку на поток, из которого был вызван
